# OrbChase 5-Minute Demo Video Script

**Title:** OrbChase: Building a Real-Time 2D Multiplayer Game in One Day with AI

**Length:** ~5 minutes  
**Tone:** Confident, energetic, educational + fun  
**Structure:** Gameplay → Tech Walkthrough → AI Reflection

## 0:00 - 0:45 | Hook & Intro (Gameplay Montage)
- Fast cuts: Multiple browser tabs joining a room, players dashing and collecting orbs, hazard spinning, scores climbing, level-up screen.
- Voiceover: "In one day, using AI as my learning engine, I built OrbChase — a real-time 2D multiplayer arena game that meets every requirement from the Game Week brief for Path 2."
- Show title screen + lobby with arena unlocks.

## 0:45 - 2:15 | Core Gameplay Demo (Fun Factor)
- Play a 60-second match with 2-3 "friends" (other tabs or recorded).
- Narrate controls: "WASD to move, SPACE to dash with cooldown and trail. Collect golden orbs while dodging the rotating red hazard."
- Highlight: Real-time positions of other players updating live, orb respawn, score pop, hazard damage flash.
- End match → Results screen with XP bar and unlock text.
- "Three arenas unlocked by level, persistent progress via localStorage."

## 2:15 - 3:30 | Technical Walkthrough
- Screen share code:
  - `GameScene.js`: Phaser Arcade Physics, geometric player (circle + emoji), hazard rotation + collision, overlap collect.
  - `SocketManager.js`: io connection, room join, throttled move emits, other player lerp reconciliation.
  - `server/index.js`: Express + Socket.io rooms Map, 20Hz broadcast, event handling.
- Explain: "Client-side prediction for instant feel + server reconciliation for accuracy. Low-latency even on free tiers."
- Show package.json, render.yaml, and how it deploys to Vercel (static) + Render (Node).

## 3:30 - 4:30 | AI Utilization & Learning Velocity
- Show PRD.md and BRAINLIFT.md.
- "Started with a detailed PRD generated by AI that mapped every ClientRequirements deliverable to a 1-day plan."
- Highlight key prompts: architecture decisions, Phaser patterns, Socket.io room logic, progression system.
- "AI helped me go from zero Phaser/Socket experience to production code in hours. The process is repeatable: targeted prompts → PoC → integrate → polish."

## 4:30 - 5:00 | Close & Call to Action
- Final gameplay clip + deployed URL placeholder.
- "OrbChase proves AI-augmented developers can ship polished, engaging multiplayer games in unfamiliar stacks — fast."
- "All deliverables complete: working game, GitHub repo, docs, Brainlift, demo."
- "Ready for testing and evaluation. Thanks!"

**Tips for Recording:**
- Use OBS or Loom with clean browser + code split screen.
- Background music: upbeat electronic (royalty-free).
- End screen: GitHub link + "Built in 8 hours with Cursor AI".

This script + the game itself covers the 5-minute requirement and showcases technical achievement, game quality, and AI methodology.
